1. Program by HTML5 and CSS3

2. Use support-wide-browser's js libraries, such as JQuery and knockout.js

3. Use plugins for the above js library

4. If IE's version is < 9, add html5shiv.js , respond.js , es5-shim and selectivizr.js

5. If IE's version is < 8, add json2.js

6. Use bootstrap's mixins

7. Set first meta tag

<head>
<meta http-equiv="X-UA-Compatible" content="IE=9; IE=8; IE=EDGE">

8. Set viewport for mobile phone

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width,height=device-height,initial-scale=1.0,minimum-scale=1.0,maximum-scale=1.0,user-scalable=no"/>
